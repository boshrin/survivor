About compact
basics
Module Type Scheduler Remote
format Yes No
syntax 
     format as compact

Dependencies

  • None

Arguments

None.

Description

The compact module generates a reasonably short alert message, containing only a summary and a time (as well as fix information if appropriate), suitable for display on a compact device such as a pager or mobile phone. Additionally, if a help file is available, its contents will be included at the end of the message (where they might be truncated if the destination device has message size limitations).

Examples

  1. Example compact output: 
         Test is not responding.
     18:31:30 02/14/2003
