About swraid
basics
Module Type Scheduler Remote
check No Yes
syntax 
     module swraid {
       [ignore <string>]
       [ignore ...]
     }
     via <transport>

Dependencies

  • None

Arguments

Name Type Value
ignore optional string list List of raid devices to ignore.

Description

The swraid module checks the status of Linux software raids as determined by examining /proc/mdstat.

Examples

  1. To test swraid: 
         module swraid {}
     via remote
