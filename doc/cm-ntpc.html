About ntpc
basics
Module Type Scheduler Remote
check Yes No
syntax 
     module ntpc {}

Dependencies

  • Executables
    • ntpq

Arguments

None.

Description

The ntpc module determines the validity of the time peers on the host to be checked by executing ntpq -p hostname. If the module cannot reach the ntp client, the client has fewer than one peer, or all peers are unreachable (stratum 16), a problem is generated. If only some, but not all, peers are unreachable, a warning is generated.

Examples

  1. To test ntpc: 
         module ntpc {}
