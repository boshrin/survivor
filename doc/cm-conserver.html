About conserver
basics
Module Type Scheduler Remote
check Yes Yes
syntax 
     module conserver {}
     [via <transport>]

Dependencies

  • Executables
    • console

Arguments

None.

Description

The conserver module uses Conserver's console utility to check the status of its consoles. If any console is marked down, a PROBLEM is returned.

Examples

  1. To test for down consoles: 
         module conserver {}
     via remote
