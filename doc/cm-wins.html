About wins
basics
Module Type Scheduler Remote
check Yes Yes
syntax 
     module wins {
       name <string>
     }

Dependencies

  • Executables
    • nmblookup

Arguments

Name Type Value
name string NetBIOS name or workgroup to look up.

Description

The wins module uses the nmblookup utility from the Samba package to query the WINS server in order to determine if the server is returning expected results.

Examples

  1. To test WINS: 
         module wins {
       name someHost
     }

     module wins {
       name otherWorkGroup
     }
