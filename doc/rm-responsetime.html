About responsetime
basics
Module Type Scheduler Remote
report Yes No
check output format When used with the report check module, the output provided to the module will be one column: 
     averageduration

Dependencies

  • None

Arguments

None.

Description

The responsetime module determines the average execution duration for the provided data.

The summary module utilizes check records.

Examples

  1. Example reporting output: 
         http@server1: 70.25ms
  2. Example check formatted output: 
         70.25
