About smbd
basics
Module Type Scheduler Remote
check Yes No
syntax 
     module smbd {}

Dependencies

  • Executables
    • smbclient

Arguments

None.

Description

The smbd module uses the smbclient utility from the Samba package to contact the SMB server and query it for a list of available shares in order to determine if the server is returning expected results.

Examples

  1. To test the SMB server: 
         module smbd {}
