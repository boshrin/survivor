About telnet
basics
Module Type Scheduler Remote
check Yes No
syntax 
     module telnet {
       [port <number>]
       [prompt <string>]
     }

Dependencies

  • Perl Modules
    • Net::Telnet

Arguments

Name Type Value
port optional number, default 23
prompt optional string, default is 'login:' Prompt to search for.

Description

The telnet module connects to a telnet server and searches for a specified login prompt in order to determine is the server is returning expected results.

Examples

  1. To look for 'login:' on port 23: 
         module telnet {}
  2. To look for 'login:' on port 999: 
         module telnet {
       port 999
     }
  3. To look for 'user:' on port 23: 
         module telnet {
       prompt user:}
