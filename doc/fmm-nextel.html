About nextel
basics
Module Type Scheduler Remote
format Yes No
syntax 
     format as nextel

Dependencies

  • None

Arguments

None.

Description

The nextel module generates a message designed to enable reply commands from Nextel two-way messaging capable devices. The mail gateway must also be configured to use this service. The module will generate a reasonably short alert message, containing only a summary and a time (as well as fix information if appropriate), suitable for display on a nextel device such as a pager or mobile phone. Additionally, if a help file is available, its contents will be included at the end of the message (where they might be truncated if the destination device has message size limitations).

Examples

  1. Example nextel output: 
         Test is not responding.
     18:31:30 02/14/2003
