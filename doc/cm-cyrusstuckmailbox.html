About cyrusstuckmailbox
basics
Module Type Scheduler Remote
check No Yes
syntax 
     module cyrusstuckmailbox {}
     via <transport>

Dependencies

  • Executables
    • ctl_mboxlist

Arguments

None.

Description

The cyrusstuckmailbox module executes ctl_mboxlist -mw to determine if there are any mailboxes that have not yet been synchronized with the murder master and returns an error if found.

Examples

  1. To test for stuck mailboxes: 
         module cyrusstuckmailbox {}
     via remote
