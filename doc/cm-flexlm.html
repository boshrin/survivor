About flexlm
basics
Module Type Scheduler Remote
check Yes No
syntax 
     module flexlm {
       license <file>
     }

Dependencies

  • Executables
    • lmstat

Arguments

Name Type Value
license file License file for FLEXlm server.

Description

The flexlm module connects to a Flexible License Manager (FLEXlm) server on the host to be checked and confirms that the license server is up and running. The module uses the lmstat utility from the FLEMlm package to contact the FLEXlm server and determine if the server is returning expected results.

Examples

  1. To test flexlm: 
         module flexlm {
       license /etc/software.dat
     }
