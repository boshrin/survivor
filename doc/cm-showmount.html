About showmount
basics
Module Type Scheduler Remote
check Yes No
syntax 
     module showmount {}

Dependencies

  • None

Arguments

None.

Description

The showmount module uses the showmount executable to determine if the host being checked is correctly exporting filesystems. If the module cannot obtain the export list for the host, a problem is generated. If it can obtain the export list, but no filesystems are exported, a warning is generated.

Examples

  1. To test exported filesystems: 
         module showmount {}
